# WiresharkDNS
Lab 1 for [Mike Wittie](https://www.cs.montana.edu/mwittie/)'s Fall 2017 Networks course.

**Assignment**
In this assignment you will learn to use Wireshark traffic monitor to gain a deeper understanding of DNS. Besides working with DNS, knowing how to use Wireshark is an essential skill in administration, network experimentation, and distributed system implementation. The assignment is composed of two labs.
